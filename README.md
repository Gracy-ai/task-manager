# Task Manager Application

Application de gestion de tâches développée avec Java Spring Boot et MongoDB Atlas.

## Technologies utilisées
- Java Spring Boot
- MongoDB Atlas
- HTML/CSS/JavaScript
- Maven

## Prérequis
- Java 17 ou supérieur
- Maven (ou utilisez le wrapper inclus)
- Compte MongoDB Atlas
